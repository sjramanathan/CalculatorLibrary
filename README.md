# Calculator Library

## Aim: 
Learn how to make a Nuget package and import it into a HelloWorld .NET application.

## Technology Stack: 
  - Framework: .NET Core 2.1
  - Language: C#
  
## Project Output: 
Build a simple calculator library with the functions add, subtract, multiply and divide.
